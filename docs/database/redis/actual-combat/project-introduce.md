# Redis实战 - 项目介绍

## 项目介绍

本项目是黑马程序员发布的黑马点评网项目，项目基于Spring Boot + Redis + MySQL 技术栈开发。

功能模块：
1. 短信登录
2. 商户查询缓存
3. 达人探店
4. 优惠券秒杀
5. 好友关注
6. 附近的商户
7. 用户签到
8. UV 统计
